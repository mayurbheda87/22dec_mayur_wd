(2) What are tags and attributes in HTML?

Ans: HTML tags are the keywords on a web page that define how your web browser
must format and display your web page. Almost all tags contain two parts,
an opening, and a closing tag. For example,
<html> is the opening tag and </html> is the closing tag.
AND

An HTML attribute is a piece of markup language used to adjust the behavior or display
of an HTML element.

EX. attributes can be used to change the color, size, or functionality of HTML elements.

EX.<a attribute-name:- href="value">Content</a>
