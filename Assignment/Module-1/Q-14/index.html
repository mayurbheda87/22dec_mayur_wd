(14) What is the use of a span tag? Explain with example?

Ans:
The span tag is a paired tag means it has both open(<) and closing (>) tags,
and it is mandatory to close the tag.
The span tag is used for the grouping of inline elements & this tag
does not make any visual change by itself.
span is very similar to the div tag, but div is a block-level tag and span is an inline tag.

Example:

<span class="">Some Text</span>
