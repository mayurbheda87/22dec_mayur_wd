(1) Are the HTML tags and elements the same thing?

Ans:-No, HTML tags and elements are not same thing.

HTML tag starts with < and ends with >   

AND

HTML elements is The content between the start tag and the end tag is called an element.

Example of HTML Tag: <b> </b>    <p> </p>

Example of HTML Elements: <b>This is the content.</b>   <p>This is the content.</p>
