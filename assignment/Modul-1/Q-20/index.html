(20) What are logical and physical tags in HTML?

Ans:
Logical Tags :
Logical Tags are used in HTML to display the text according to the logical styles. Following are
the Logical tags commonly used in HTML.

Example of Logical tag:

<abbr>-- Defines an abbreviation

<strong>-- Defines strong text

<var>-- Defines a variable

<address>-- Defines an address element

Physical Tags:
Physical Tags are used in HTML to provide actual physical formatting to the text. Following are
the Physical tags =commonly used in HTML.

Example of Physical Tags:

<b>-- Defines bold text

<big>-- Defines big text

<i>-- Defines italic text

<small>-- Defines small text
