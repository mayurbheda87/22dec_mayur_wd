(7) What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements?

Ans:The difference between Class and ID:
A Class name can be used by multiple HTML elements,
while an ID name must only be used by one HTML element within the page.

Example of ID:

The id attribute is written using the # symbol followed by id.

id='id_name'

The class name in CSS stylesheet using “.” symbol.

class='class_name'
