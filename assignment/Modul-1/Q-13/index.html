(13) What is the use of an iframe tag?

Ans:An iFrame, also knowns as Inline Frame, is an element that loads another HTML element
inside of a web page. They are commonly used to embed specific
content like external ads, videos, tags, or other interactive elements into the page.

Advantages:

1)easy to use
2)well supported
3)allows embedding of third-party content

Disadvantages:

1)memory and processor heavy
2)no control of third-party sites
3)difficult to use in responsive design
4)may impact SEO
