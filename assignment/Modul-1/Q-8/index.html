(8) What are the various formatting tags in HTML?

Ans:HTML provides many predefined elements that are used to change the formatting of text.
The formatting can be used to set the text styles (like â€“ bold, italic, or emphasized,),
highlighting the text, making text superscript and subscript, etc.

Example of HTML formatting Tag:

Element name Description
<b> This tag is used to bold the text written between it.
<strong> This tag is tells the browser that the text is important.
<i> This tag is used to make text italic.
<mark> This tag is used to highlight text.
<small> This tag is used to decrease the font size by one unit from base font size.
<big> This tag is used to increase the font size by one conventional unit.
<u> This tag is used to underline text written between it.
<sup> It displays the content slightly above the normal line.
<sub> It displays the content slightly below the normal line.
